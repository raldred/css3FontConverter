Converts TTF (OTF) to other web formats, woff, svg, eot & a prepared stylesheet

    $ ./convertfonts *.(otf|ttf)

Requirements:

	* Font Forge
	`brew install fontforge`

	* ttf2eot
	`brew install ttf2eot`

	* fondu
	`brew install fondu`

	* sfnt2woff (included - put it somewhere in your path)
	http://people.mozilla.org/~jkew/woff/


Originally based on https://github.com/zoltan-dulac/css3FontConverter
